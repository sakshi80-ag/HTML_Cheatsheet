  2. Headings

    There are six headings available in HTML, H1 is the largest among all, and H6 is the smallest.

    <h1> Tag

    <h1>Heading 1</h1>

    <h2> Tag

    <h2>Heading 2</h1>

    <h3> Tag

    <h3>Heading 3</h1>

    <h4> Tag

    <h4>Heading 4</h1>

    <h5> Tag

    <h5>Heading 5</h1>

    <h6> Tag

    <h6>Heading 6</h1>

  3. <div> tag

    div tag or division tag is used to make blocks or divisions in the document. 
     <div> This is div block </div>

  4. <span> tag

    span is a container for inline content 
    <span> This is span block </span>

  5. <p> tag

    Paragraph 
    <p> This is a paragraph </p>

  6. <b> tag

    Bold 
    <b> This is bold text </b>

  7. <strong> tag

    Strong 
    <strong> This is strong text </strong>

  8. <em> tag

    Emphasized 
    <em> This is emphasized  text </em>

  9. <i> tag

    Italic 
    <i> This is italic text <i>

  10. <ol>tag

    Ordered list starts with <ol> tag and each list item starts with <li> tag Order List 
    <ol>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ol>

  11. <ul> tag

    UnOrdered list starts with <ul> tag and each list item starts with <li> tag Order List 
    <ul>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ul>

  12. <audio> tag

    It is used to embed sound content in the document. 
    <audio controls>
<source src="demo.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

  13. <img> tag

    It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">

  14. <video> tag

    It is used to embed video in the webpage. 
    <video width="480" height="320" controls>
<source src="demo_move.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>

  15. Table

    A table is a collection of rows and columns. It is used to represent data in tabular form. 
    <table>
<caption>Demo Table</caption>
<thead>
<tr>
<th>Column1</th>
<th>colspan="2">Column2</th>
</tr>
</thead>
<tbody>
<tr>
<td>Data1</td>
<td>Data2</td>
<td>Data2</td>
</tr>
<tr>
<td>Data1</td>
<td>Data2</td>
<td>Data2</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>&nbsp;</td>
<td>Data</td>
<td>Data</td>
</tr>
</tfoot>
</table>

  16. <a> tag

    Anchor tag defines a hyperlink. 
    <a href="www.google.com/">Visit google!</a>
